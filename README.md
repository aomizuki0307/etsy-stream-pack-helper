# Etsy Stream Pack Helper

Seller-only tool for automating Etsy-ready digital stream overlay packs.

* Generates listing images (JPG) and download ZIPs.
* Creates/updates draft listings via Etsy Open API (seller review before publish).
* No buyer data collected; no messaging; no scraping.

## What it does

* Batch-generate overlay assets (PNG, JPG) and README.txt
* Prepare 4â€“5 digital download ZIPs per listing
* Optional: auto-create listing drafts with placeholders

## What it does NOT do

* No buyer PII access, no order data, no messaging
* No scraping; only official Etsy API calls
* No third-party seller access (this is for my own shop)

## Data handling

* Stores only listing assets/metadata
* Does not store buyer information
* Caching follows Etsy API Terms
